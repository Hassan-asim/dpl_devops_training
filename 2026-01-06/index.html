EC2 Static Website Hosted Successfully!

Using AWS EC2, Nginx, and Let’s Encrypt SSL

Steps Completed

1. EC2 Instance Launched

Created an Amazon Linux 2 EC2 instance to host the static website.

2. Connect via SSH

Used a key pair file to connect to the EC2 instance securely from a local Windows laptop.

3. Install Nginx

Installed Nginx web server on EC2 to serve the static website content.

4. Upload Website Files

Uploaded index.html and error.html to /usr/share/nginx/html.

5. Point Domain

Configured the domain DNS A record to point to the EC2 public IP for the website.

6. SSL with Let’s Encrypt

Installed Certbot and generated a free SSL certificate. HTTPS is now enabled with auto-renewal.

7. Tested Website

Verified website loads correctly on http and https, both locally and on VPN.

Technologies & Tools

Amazon EC2

Elastic Compute Cloud (EC2) provides scalable virtual servers to host websites and applications.

Nginx

Lightweight, high-performance web server used to serve static content and handle HTTP/HTTPS requests.

Certbot & Let’s Encrypt

Free SSL certificate management tool that provides HTTPS for secure connections.

DNS / Domain

Configured domain A record to point to EC2 instance for proper website access.
