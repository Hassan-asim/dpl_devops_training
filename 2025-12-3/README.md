## üî∑ TypeScript Fundamentals

I started my TypeScript journey by going through the official handbook. My focus was on the foundational concepts that underpin the language.

### 1. The Basics & Everyday Types
From the [official handbook](https://www.typescriptlang.org/docs/handbook/2/basic-types.html), I covered:
- **Core Basic Types**: `string`, `number`, and `boolean`.
- **Arrays**: How to define typed arrays (e.g., `number[]`).
- **`any`**: The escape hatch from the type system.
- **Type Annotations on Variables**: Explicitly setting types with `: type`.
- **Functions**: Typing parameters and return values.
- **Object Types**: Defining the shape of objects with property names and types.
- **Union Types**: Combining types with the `|` operator (e.g., `string | number`).
- **Type Aliases**: Creating custom names for types using `type`.
